# Lince Zed Theme

A minimal black-and-white theme for [Zed](https://zed.dev).
No gradients. No distractions. Just contrast.

## Install from Zed Store
1. Open **Extensions** (Ctrl+Shift+X)
2. Search for **Lince**
3. Click **Install**
4. Set your theme:
   ```json
   {
     "theme": "Lince Dark"
   }
